Hexo使用

###　第一步安装
npm install -g hexo-cli

第二步：建站

hexo init <folder>
cd <folder>
npm install

写文章

hexo new [layout] <title>
hexo new draft "测试"

写草稿

hexo new draft "测试draft"      写草稿
hexo publish draft "测试draft"  发布草稿

配置文件在_config.yml 中查看
